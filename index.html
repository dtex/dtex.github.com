Not Open Source Projects

SPAS

Super Proxy Asset Server is still in beta, but I'm pretty excited about it. SPAS is a node.js service that concatenates, caches and throttles multiple API lookups. API responses can be parsed by custom functions in the service. For example, we had a client whose Media page made a request to YouTube for videos in their feed, Twitter for their recent tweets, and eleven requests to Flickr for a list of photo sets and photos in each set. SPAS combines them all into a single request and eliminates a large portion of the client side parsing. A huge part of the credit for SPAS goes to neopunisher
